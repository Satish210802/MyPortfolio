WECARE: FOOD RECCOMENDATION SYSTEM

Key skills & Tools: HTML | CSS | JAVASCRIPT | PYTHON | ML

Developed a web-based diet recommendation system that analyzes user preferences, health goals, and nutritional requirements to provide personalized diet plans and recommendations. The system prioritizes participantâ€™s actual willingness to change body behavior, ensuring tailored recommendations for effective dietary adjustments. Presented findings at the GCITC IEEE conference.
The system utilizes k-means clustering to divide the dataset into different clusters based on factors like diet type and calorie range. This clustering process helps divide the user dataset into distinct groups, allowing for more targeted and specific recommendations. Subsequently, by applying the random forest algorithm to clusters with parameters such as age, BMI, and calorie intake, the system generates a personalized three-meal diet plan tailored to the user's body state and daily calorie goals. Users have the flexibility to select food items that align with their calorie targets and preferences, ensuring a practical and sustainable approach to diet planning. Moreover, the system enables users to save their customized meal plans for future reference, promoting consistency and continuity in their nutritional journey.
